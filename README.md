# VKR_COLD

Задача (Цель):   
Целью проекта является разработка и технико-экономическое обоснование проектных решений для хладоцентра ЦОД (центра обработки данных), обеспечивающих минимальное энергопотребление. Исследование сфокусировано на применении современных гибридных схем охлаждения и технологий фрикулинга (свободного охлаждения), адаптированных к климатическим условиям Санкт-Петербурга

Для достижения цели были выполнены следующие шаги:   
Анализ технологий: Проведен обзор современных систем охлаждения ЦОД, включая жидкостное и гибридное охлаждение.  Климатическое моделирование: Исследовано влияние климата Санкт-Петербурга на эффективность оборудования с использованием метода температурных бинов (на основе СП 131.13330.2020).  Сравнительный расчет: Разработана методика и выполнено сопоставление трех схем хладоцентра для объекта мощностью 1 МВт ИТ-нагрузки:  Традиционная фреоновая схема (базовая).  Гибридная схема с фрикулингом (рекомендуемая).  Гибридная схема с фрикулингом и тепловым аккумулятором холода (ТАХ).  Инструментарий: Расчеты помесячного энергопотребления, часов фрикулинга и показателей NPV реализованы на Python 3.11 (NumPy, SciPy, pandas, matplotlib), расчеты CAPEX/OPEX — в Microsoft Excel

Выводы   
Энергоэффективность:   
Рекомендуемая гибридная схема со свободным охлаждением (Схема 2) обеспечивает коэффициент PUE = 1,310.  Экономия: Применение фрикулинга позволяет снизить затраты на электроэнергию для системы охлаждения на 43,5% (общее потребление ЦОД снижается на 12,3%).  Окупаемость: Простой срок окупаемости дополнительных вложений составил всего 1,11 года.  Оптимизация: Показано, что усложнение системы (Схема 3 с ТАХ) в условиях Санкт-Петербурга экономически менее выгодно, чем стандартная схема с фрикулингом, из-за высоких капитальных затрат при незначительном приросте эффективности
